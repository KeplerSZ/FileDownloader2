# FileDownloader2

Консольное приложение для автоматического скачивания файлов.

## Как запустить:
1. Скачайте проект
2. Заполните `FileSettings.json` своими данными
3. Запустите: `dotnet run`

## Что делает:
- Проверяет доступность сайта
- Авторизуется (Basic Auth)
- Ищет последний файл для скачивания
- Скачивает файл
- Пишет логи в папку `logs/`

## Технологии:
- .NET 10.0
- C#
